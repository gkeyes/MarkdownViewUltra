# Markdown View Ultra

A lightweight Markdown previewer APK for Android.

## Features

- Open `.md` files via Android "Open with" from any file manager
- Beautiful Markdown rendering with Material Design 3
- Dark/Light theme (follows system)
- Clickable links
- Pure preview - no editing, ultra lightweight

## Build

This project uses GitHub Actions to build the APK automatically.

1. Push to `main` branch
2. Go to Actions tab
3. Download the APK artifact

## Manual build

```bash
flutter pub get
flutter build apk --debug
```
